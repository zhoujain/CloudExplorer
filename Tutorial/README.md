# CloudExplorer-Lite 学习教程

> 本教程基于对仓库源码的梳理，帮助你由浅入深地理解并上手 CloudExplorer-Lite（以下简称 CE-Lite）这个开源云管平台。

## 这是什么项目

CE-Lite 是飞致云（FIT2CLOUD）开源的**轻量级多云管理平台**，能纳管阿里云、腾讯云、华为云、VMware、OpenStack、Proxmox VE 等主流公有云/私有云，开箱提供云主机管理、云账单、运营分析、安全合规等功能，并具备多租户体系与模块化扩展能力。
